# WeatherLy - Real-Time Forecast & Recommendation Engine

A clean, responsive weather application that goes beyond simple temperature metrics to provide context-aware daily lifestyle and activity recommendations based on live atmospheric data.

## Features
- **Asynchronous API Integration:** Fetches global, real-time weather data dynamically upon city submission.
- **Contextual Recommendation Engine:** Programmatically parses current weather descriptions (e.g., clear sky, moderate rain) to output matching activity prompts like café visits or indoor hobbies.
- **Polished Glassmorphism UI:** Features frosted glass cards, smooth error notifications for blank inputs, and fluid text transitions overlaid on a minimalist cinematic background.

## Tech Stack
- HTML5
- CSS3 (Glassmorphic Backdrop Filters, Absolute Positioning, Flexbox)
- JavaScript (Fetch API, Promises, Dynamic DOM Rendering)
